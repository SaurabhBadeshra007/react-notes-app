# 📝 React Notes App

A modern and responsive Notes Management application built with **React**, **Vite**, and **Tailwind CSS**.

This project demonstrates CRUD operations, React state management, component-based architecture, form validation, and deployment using GitHub & Netlify.

# 📖 Project Overview

React Notes App is a beginner-friendly yet scalable project built to practice modern frontend development.

Users can create, edit, update, and delete notes with a clean, responsive interface. The project follows good Git practices with meaningful commit history and will continue to evolve with advanced features like authentication, search, categories, Firebase integration, and more.

# 📥 Installation

Clone the repository

```bash
git clone https://github.com/SaurabhBadeshra007/react-notes-app.git
```

Move into the project

```bash
cd react-notes-app
```

Install dependencies

```bash
npm install
```

Start Development Server

```bash
npm run dev
```

Build Production

```bash
npm run build
```

Preview Production Build

```bash
npm run preview
```
